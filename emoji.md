
# emoji commit template

| image  | emoji        | mean           |
| :----: | ------------- |:-------------|
| 🎉     | `:tada:` | new big features |
| 👍     | `:+1:` | new features |
| 🚿     | `:shower:` | delete |
| 🆕     | `:new:` | add new files |
| 🐛     | `:bug:` | bugfix |
| ♻️      | `:recycle:` | refactoring |
| 📝     | `:memo:` | add comments |
